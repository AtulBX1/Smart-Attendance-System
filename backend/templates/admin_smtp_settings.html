📧 Configure Email Delivery

Set up SMTP to send OTP emails directly to faculty. Without this, OTPs will print to the server console.

📝 Gmail Setup Guide:

1. Use smtp.gmail.com as the host, port 587
2. Go to Google App Passwords
3. Generate an "App Password" for "Mail"
4. Use that 16-character password here (not your regular Gmail password)

Note: You must have 2-Step Verification enabled on your Google account.
